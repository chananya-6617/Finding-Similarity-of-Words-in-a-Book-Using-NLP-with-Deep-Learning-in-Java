# 📚 Finding Similarity of Words in a Book Using NLP with Deep Learning in Java

This project is a Java-based Natural Language Processing (NLP) application that identifies semantically similar words from a given book using the **Word2Vec** algorithm. Built with **Deeplearning4j (DL4J)** and **ND4J**, the application trains a word embedding model on the book's text and learns the contextual relationships between words. Users can enter any word from the book, and the system returns the most similar words based on their learned vector representations.

## ✨ Features

* Reads and processes text from a book.
* Performs text preprocessing and tokenization.
* Trains a Word2Vec model using Deeplearning4j.
* Finds semantically similar words based on context.
* Displays the top similar words with similarity scores.
* Demonstrates practical Natural Language Processing and Deep Learning concepts in Java.

## 🛠️ Technologies Used

* Java
* Deeplearning4j (DL4J)
* ND4J
* Maven
* Word2Vec
* Natural Language Processing (NLP)

## 🚀 How It Works

1. Load the text from a book.
2. Preprocess and tokenize the text.
3. Train the Word2Vec model.
4. Enter a word to search.
5. View the most similar words generated by the model.

This project demonstrates how deep learning techniques can be applied to Natural Language Processing tasks by learning meaningful word relationships from textual data. It provides a practical understanding of word embeddings, semantic similarity, and AI-based text analysis using Java.
